# TASK-RUNNER

Un programma cli per eseguire script automaticamente(simile a npm)

## Requisiti:
- task-runner

## Avviare:
  1. **Senza opzioni**
  ```bash
  task-runner [nome_script]
  ```

  2. **Output file**
  ```bash
  task-runner [nome_script] -o <file>
  ```

  3. **Verbose**
  ```bash
  task-runner [nome_script] -v
  ```
